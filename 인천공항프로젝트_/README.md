# 🛫 인천공항 프로젝트 (Incheon Airport Project)

> **데이터 기반으로 인천국제공항 5단계 확장 타당성을 입증한 분석 프로젝트**  
> Python · Selenium · Plotly · OAG Data 기반 공항 허브 연결성 및 화물 운항 분석  

---

## 📌 프로젝트 개요

**목표:**  
인천국제공항은 2033년 포화가 예상됨에 따라 제3여객터미널과 제5활주로 신설을 포함한 5단계 확장 계획을 추진 중입니다.  
본 프로젝트는 인천공항 측이 국토교통부를 설득하기 위해 데이터 기반 근거를 마련하고자 KMAC에 의뢰한 분석 과제입니다. 

---

## 🧩 나의 역할

| 구분 | 주요 내용 |
|------|------------|
| 📊 Hub Connectivity 분석 | - OAG 데이터를 활용한 허브 연결성 지표산출<br>- Python 기반 공항별 연결성 비교 |
| 🤖 데이터 크롤링 자동화 | - Selenium 기반으로 공항 데이터 관련 플랫폼의 110개 공항 데이터 자동 수집 시스템 구축을 통한 데이터 수집 시간 단축|
| 📈 시각화 및 리포팅 | - Plotly를 활용한 공항별 분석 결과 시각화 대시보드(HTML) 제작|
| 🕒 화물 운항 분석 | - 화물 운항 패턴을 분석하여 활주로 확장 필요성 정량화|

---

- **Language:** Python 3.10  
- **Libraries:** pandas, numpy, plotly, selenium, BeautifulSoup, geopy  
- **Data Source:** OAG Schedule Data
- **Visualization:** Plotly  
- **Tools:** VSCode, Excel
